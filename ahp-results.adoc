== Evaluacion AHP de Stacks Tecnologicos

=== Objetivo
Seleccionar el stack tecnologico mas conveniente para ReflectAI entre:

* A: Next.js + React + TypeScript + Tailwind CSS + Supabase PostgreSQL + Groq API + Vercel
