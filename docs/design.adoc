== Modelo de datos (diagrama de entidad-relación)
Diseño conceptual de la base de datos (NoSQL / Relacional).

[mermaid]
....
erDiagram
    auth_users ||--|| profiles : "crea perfil (trigger)"
    profiles ||--o{ reflection_sessions : "registra"

    auth_users {
        uuid id PK "Gestionado por Supabase Auth"
        varchar email
        varchar encrypted_password
    }

    profiles {
        uuid id PK "FK -> auth.users.id"
        varchar full_name
        timestamp created_at
        timestamp updated_at
    }

    reflection_sessions {
        uuid id PK
        uuid user_id FK "FK -> profiles.id"
        timestamp started_at
        timestamp completed_at
        varchar status "enum: 'draft', 'completed'"
        varchar title "Generado por IA al final"
        jsonb payload "Estructura dinámica de preguntas/respuestas"
        jsonb ai_analysis "Métricas pre-calculadas para el Dashboard"
    }
....
